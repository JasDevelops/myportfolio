Chatterbox

A React Native Chat App with Real-Time Messaging

Chatterbox is a mobile chat application built using React Native, Expo, and Firebase. It allows users to send text messages, images, audio, and share their location in real-time, with offline support for message storage.

Project Objective

The goal of Chatterbox is to provide a seamless chat experience with features like:

  • Real-time messaging with Firestore
  • Offline message storage using AsyncStorage
  • Support for sending images, audio messages, and location sharing
  • Anonymous login via Firebase Authentication
  • Custom audio player for recorded messages

Technology Behind Chatterbox

  • React Native - Framework for cross-platform mobile development
  • Expo - Simplified development and testing
  • Firebase Firestore - Real-time cloud database
  • Firebase Authentication - Secure user login
  • Gifted Chat - Library for chat UI components
  • AsyncStorage - Local storage for offline functionality

Challenges & Growth

Developing Chatterbox marked my first foray into mobile app development, a domain increasingly vital in today's tech landscape. Mastering React Native and ensuring smooth functionality across both Android and iOS was a significant learning experience. A major challenge emerged during testing when I encountered storage limitations while running both an Android emulator and an iOS simulator simultaneously. Managing system resources efficiently became crucial, reinforcing the importance of optimizing development environments for mobile testing. Overcoming these constraints not only expanded my technical problem-solving skills but also deepened my understanding of the practical challenges in mobile application development.
