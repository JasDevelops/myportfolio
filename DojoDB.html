DojoDB-client

The ultimate Martial Arts Movie Database

DojoDB is a full-stack movie database application designed for martial arts movie enthusiasts. It consists of a React-based frontend and a RESTful API backend, allowing users to browse, search, and manage their favorite movies. The app provides a seamless experience, enabling users to filter movies, explore details about actors and directors, and manage their personal profiles.

Project Objective

DojoDB-client is built to provide an engaging and user-friendly platform where movie lovers can:

  • Browse and search a curated collection of martial arts movies
  • View detailed information about movies, directors, and actors
  • Filter movies by title, genre, release year, actor, or director
  • Create an account, manage their profile, and maintain a personal favorites list
  • Enjoy a clean, responsive UI with intuitive navigation

Technology Behind DojoDB

Frontend (DojoDB-client)

  • React - For a dynamic and interactive user experience
  • React Router - For seamless navigation
  • Redux - For efficient state management
  • Bootstrap 5.3 & SASS - For responsive styling and UI components
  • Parcel - Fast and efficient bundling
  • Netlify - For easy frontend deployment

Backend (DojoDB-API)

The frontend connects to DojoDB API, a RESTful backend that provides real-time movie data and handles user authentication. It is built using:

  • Node.js & Express - For handling API requests
  • MongoDB & Mongoose - For storing and managing movie/user data
  • JWT & Passport.js - For secure authentication
  • CORS - To enable cross-origin API requests
  • MongoDB Atlas - Cloud-based database storage
  • Heroku - For backend deployment

Challenges & Growth

The development of DojoDB was a pivotal learning experience. While I enjoyed crafting the frontend, the testing phase presented unexpected challenges. I encountered a compatibility issue between Puppeteer and the application, which disrupted the end-to-end testing cycle. This obstacle required me to delve deeper into debugging and testing strategies, ultimately enhancing my problem-solving skills and resilience.
