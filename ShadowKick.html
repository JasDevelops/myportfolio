ShadowKick

An Angular-based Movie App

ShadowKick is a web application built with Angular that allows users to browse a collection of movies, learn more about their details, and manage personal favorites. It provides a seamless and interactive experience for movie lovers.

Project Objective

The goal of ShadowKick is to provide users with an intuitive movie browsing platform that allows them to:

  • Register and log in securely
  • Browse a collection of movies
  • View detailed information about movies, directors, and genres
  • Manage a list of favorite movies
  • Experience a responsive and user-friendly interface

Technology Behind ShadowKick

  • Angular - Framework for dynamic web applications
  • Angular Material - UI components for a responsive and accessible design
  • TypeScript - For code maintainability
  • RESTful API - Connects to a backend for real-time movie data
  • JWT Authentication - Secure user login and authentication
  • GitHub Pages - For deployment and hosting

Challenges & Growth

Developing ShadowKick was an enjoyable experience and progressed more smoothly compared to previous projects like DojoDB. This project allowed me to deepen my understanding of Angular’s capabilities, enhancing my proficiency in building dynamic web applications. Additionally, I explored TypeScript for improved code maintainability and leveraged TypeDoc to generate comprehensive documentation. However, as someone new to Angular, I found its official documentation challenging to navigate, making the learning process more difficult than expected. Despite this, overcoming these hurdles strengthened my adaptability and problem-solving skills, further solidifying my grasp of these technologies.
