Eventify

Your Smart Event Companion

Eventify is a modern, serverless Progressive Web App (PWA) designed to help users discover and track events effortlessly. Eventify provides a seamless experience with both online and offline access.

Project Objective

The goal of Eventify is to make event discovery simple and accessible anytime, anywhere. Built with a Test-Driven Development (TDD) approach, the app offers features like:

  • Search for events by city
  • View event details & manage how many events are displayed
  • Offline functionality - access cached events without an internet connection
  • Add the app to your home screen for quick access
  • Interactive charts to visualize event trends by city & category

Technology Behind Eventify

Eventify is lightweight, easy to use, and works across devices, making event discovery more convenient than ever. To ensure smooth performance and reliability, Eventify is built using:

  • React - A fast, responsive front-end framework
  • AWS Lambda - A serverless backend for handling data requests
  • Google Calendar API - Fetching real-time event data
  • Jest & Cucumber - Ensuring quality through rigorous testing
  • OAuth2 - Secure authentication
  • Recharts - For beautiful data visualizations
  • Service Workers - Enabling offline access
  • Vercel - For deployment

Challenges & Growth

Developing Eventify was a transformative experience in mastering serverless architectures and Progressive Web Apps (PWAs). One of the most enlightening aspects was implementing offline functionality with service workers, enabling the app to cache assets and maintain seamless access even in the absence of an internet connection. This project reinforced my commitment to building resilient applications that deliver uninterrupted, high-quality user experiences, regardless of network conditions.
