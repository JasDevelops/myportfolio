MiniMon

Gotta Code 'Em All!

MiniMon is a fun and interactive JavaScript application that lets users explore a wide range of Pokémon with just a few clicks! Powered by PokéAPI, MiniMon dynamically loads Pokémon data and presents it in a user-friendly interface with smooth navigation and responsive design.

Project Objective

MiniMon is designed to make Pokémon exploration easy and engaging while demonstrating key JavaScript functionalities. The app allows users to:

  • Browse a dynamically loaded list of Pokémon
  • View Pokémon details (name, height, and image) in a Bootstrap-powered modal
  • Search for specific Pokémon by name
  • Sort Pokémon by name or height
  • Swipe through Pokémon within the modal
  • Enjoy a responsive design with a smooth navigation bar and toggle menu

Technology Behind MiniMon

To ensure a smooth and efficient user experience, MiniMon is built using:

  • JavaScript (ES6) - Handles API requests, DOM manipulation, and event handling
  • HTML5 & CSS3 - Provides structure and styling
  • Bootstrap 5.3 - Ensures a responsive layout and interactive modals
  • PokéAPI - Fetches real-time Pokémon data
  • GitHub Pages - Hosts the application

Challenges & Growth

Building MiniMon was an exciting experience because it was my first time working on a fully functional app that connects to an API. Although I had previously built websites using WordPress and plain HTML, this project introduced me to working with real-time data and integrating APIs dynamically. It was a valuable learning experience that pushed me to think beyond static web pages and into interactive application development.
